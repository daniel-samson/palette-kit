About

A CSS UI kit with customisable color palette.

Getting Started

Add the following snippet to you page

            
            <link rel="stylesheet" href="palette-kit-theme.css" />
            <link rel="stylesheet" href="palette-kit.css" />

Update palette-kit-theme.css to the desired color


            :root {
                --palette-hue: 180;
                --palette-saturation: 50%;
              }

You can also do this in javascipt


            document.documentElement.style.setProperty("--palette-color", `hsl(${hue.value}, ${sat.value}%, 50%)`);
            document.documentElement.style.setProperty("--palette-hue", hue.value);

Backgrounds

You can set backgrounds as follows using the bg-color- prefix class 


            <div class="bg-color-500">

Text color

You can set backgrounds as follows using the text-color- prefix class 


            <div class="text-color-500">

Dark mode

Palette kit automatically detects the users preferences for dark mode and applies the colors according to their preferences. 


            <div class="bg-color-1 dark:bg-color-1 text-color-990 dark:text-color-990">

note: that dark: versions of the colors are inverted versions of each color

Typeography

The base type is 1.6rem (16px) over 1.6 line height (24px)
